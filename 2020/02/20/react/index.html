jsx

  • 注释语法 { /* 注释 */ }

  • { /* 表达式 */ } {} 大括号表示jsx语法, 括号内只要是js表达式即可, 可以是变量, 函数调用等

    { name }

  • jsx也是表达式 { <p> xxx </P> }

  • 属性

style

  • 最外层的{} 是jsx表达式 , 里面 {} 是style装样式的对象, 数值可以直接使用, 不用加单位, 一旦加单位就要用双引号
    style={ { width: “100px” } }

创建组件

  • 函数形式 { props.xx }传值

  • 引入组件 import { 组件名字 } 的组件名字与export 名字一样比较好

状态

state setState

state = { data: xxx } || constructor(props) { super(props) this.state = {xxx:xxx} } 不要直接修改 state, 使用setState({})修改

setState: 改变state的状态 setState(obj, callback) || setState(fn)

  • 在class类中使用时 去state的值 this.state.xxx 改值 this.setState({}) ,这种方法只能在该state上执行一次

  • 多次去执行setState() =>

  • this.setState(prveState => ({})) 表示返回一个对象

props 与 state的区别

1 props 不可以改变, state可以改变

2 props用于组件之间数据的传递, state用于本组件数据的使用

条件渲染和列表循环

条件渲染:

  • 三目运算符 : xx ? a : b

  • && : a && b 如果a与b都为真, 则显示b

  • 使用变量与if语句混合

事件

类似于原生js ‘ onXxx={} ‘

  • 事件函数传值 <button onClick={() => this.handleClick(值)}>添加商品 类似于原生js ‘ onXxx={} ‘

  • 也可以在constructor中 this.函数 = this.函数.bind(this),将该函数的this指向该类

  • 对象加属性可以用… 如var obj = {...obj1, name: 'aa'}

  • 父组件中使用Cart组件: <Cart data='aa'></Cart> 子组件 function Cart(props) { return( props.data ) }, 既可以使用data属性

子组件给父组件传值(尽可能让父元素去处理)

父:

子: function Cart({data, minus, add}) { <button onClick={() => minus(ele)}>- }

生命周期

1 constructor 组件构造函数执行

2 componentWillMount() {} 组件将要挂载

3 componentDidMount() {} 组件已经挂载

4 componentWillReceiveProps() {} 父组件传递的属性有变化,做相应变化 只有第二次时才会触发该钩子函数

5 shouldComponentUpdate() {} 组件是否需要更新, 需要返回布尔值的结果, 优化点

6 componentWillUpdate() {} 组件将要更新

7 componentDidUpdate() {} 组件已经更新

8 componentWillUnmount() {} 组件将要卸载

render() {} 被多次调用

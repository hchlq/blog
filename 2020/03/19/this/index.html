this的指向：函数调用时才知道this的指向

1 在全局作用域中，this指向全局对象window
2 在函数作用域中，this的指向取决于调用的对象
a.直接调用，指向全局对象window 如b();（也叫预编译）
b.通过对象（数组的下标arr[index] 或者是对象{}）的属性调用则指向该数组或对象
c.构造函数创建的对象，函数内部的this指向该新对象
3 箭头函数的this ， 没有自己的this, 找外层是否有函数，如果有，离他最近的外层函数就是它的this, 没有外层函数的话this就指向window

4 call apply 第一个参数就是它的this

一些注意事项：

1 arguements.callee 指向函数本身
2 注意区分console.log(this.name) 和 console.log(name) 前者是找对象，要找到this的指向，后者是访问对象 ，一层一层的找
3 注意在写AO和GO时要考虑this进去，同时找到this的指向
4 new 构造函数时 函数的AO中的隐式this中的变量与函数中的var 变量无关
如 function test() {
var foo = 2;
console.log(this.foo);
}
new test();
中的this.foo 是new新对象的this,与var foo 无关

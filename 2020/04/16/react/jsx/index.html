JSX

  • 定义: JSX是 facebook 起草的 JS 扩展语法，本质上是一个对象，会被 babel 编译，最终会转换为 createElement

使用的细节

每一个 JSX 表达式，有且仅有一个根节点

每个JSX元素必须有结束标记(XML规范)

  • 我们平常使用的img标签等，是可以不用结束标记的<img src='' alt=''>, 但是在React中必须要有结束标记, 可以使用或

在JSX元素中使用表达式

  • 将表达式作为内容的一部分，如果表达式是false, null或者是undefined,则不显示。普通对象不能直接放到{}中，但是React对象(如 const div = (

    111
    ))可以, 数组也可以直接放置。

  • 将表达式作为元素的属性， 在元素的属性的值，可以使用jsx表示式, 如img的src,className等

  • 属性使用小驼峰命名法， 如 marginLeft等

放置注入攻击

  • React将{}中的字符串进行编码，放置攻击

元素的不可变性

  • 虽然JXS是一个对象，但是里面所有的属性都不可以修改。
